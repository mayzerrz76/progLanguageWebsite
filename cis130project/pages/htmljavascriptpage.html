HTML/JavaScript Language

"Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript(JS)." - freecodecamp.org

In combination with one another, HTML, CSS, and JavaScript are used for front end web development. This applies to the display of a website to a client.

What is HTML & JavaScript?

HTML is :

  • Considered a markup language, as opposed to a programming language
  • Used to structure the content of a website
  • A series of elements that tell the browser how to display the webpage
  • Also supported with CSS and JavaScript to format the display of a webpage

JavaScript is:

  • A programming language that makes web pages more interactive.
  • Used to implement more complex features you see on websites
  • Used to create dynamically updated content
